What this activity is

You will learn a practical workflow for using AI in student projects without blindly trusting AI output. You will inspect a real-world food-products dataset, ask an AI assistant for help, validate the answer, and write a short decision brief.

What you need

  • A spreadsheet tool: Excel, Google Sheets, or LibreOffice Calc.
  • One AI chat assistant: ChatGPT, Gemini, Claude, or a similar tool.
  • The provided CSV file: sample_food_products.csv.

What you do not need

  • GitHub account or GitHub UI.
  • Python or coding skills.
  • Codex, Claude Code, Copilot, Cursor, or paid tools.

Beginner route

  1. Open the dataset in a spreadsheet.
  2. Read the dataset walkthrough.
  3. Use one AI assistant with the prompt pack.
  4. Check AI claims manually.
  5. Write a decision brief.
  6. Record how AI was used.

Final output

  • One-page decision brief.
  • AI-use log.
  • Validation checklist with at least three limitations or risks.
Important AI-use rule: AI can help you think, structure, and check, but you must be able to explain the final result in your own words.
