# Static site starter

This folder is a placeholder for a future static website version of the course.

The first version should still be delivered through Teams or Google Drive. A static site can be added later if the file-based course works well and a cleaner browsing experience is useful.

## Recommended first static-site approach

Start with GitHub Pages or MkDocs Material.

A minimal future site should include:

- Start here
- Beginner route
- Dataset walkthrough
- No-code AI workflow
- Validation checklist
- Decision brief template
- Prompt pack
- Examples
- Optional technical route
- Teacher notes
- Delivery notes

## Why this is not the primary delivery method

Students should not need GitHub or a website to complete the activity. The core student package should remain usable as PDFs, CSV, and an editable decision brief template.
