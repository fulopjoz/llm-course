# Start here: AI for student projects

This mini-course is designed for students who may have no coding background.

You do not need GitHub. You do not need Python. You do not need Claude Code, Codex, Copilot, Cursor, or any paid tool to finish the main activity.

The required beginner setup is:

1. A spreadsheet tool: Excel, Google Sheets, or LibreOffice Calc.
2. One AI chat assistant: ChatGPT, Gemini, Claude, or a similar tool available to you.
3. The provided CSV file: `data/sample_food_products.csv`.

## What you will learn

You will learn how to use AI in a student project without blindly trusting the output.

The workflow is:

```text
project question -> inspect data -> ask AI -> validate AI output -> write decision brief
```

## Choose your route

| Route | Best for | Required tools | Start with |

## Dataset

Use:

```text
data/sample_food_products.csv
```

Open it in Excel, Google Sheets, or LibreOffice Calc.

## Main question

The activity uses this project question:

> Based on available nutrition fields, which product category looks healthier or less healthy, and how confident can we be given missing or inconsistent data?

This is not medical advice. It is a project-management and AI-literacy exercise.

## What you submit or show at the end

Your final output is a short decision brief with:

1. Objective.
2. Data used.
3. Method.
4. Findings.
5. Limitations.
6. Recommendation.
7. AI-use log.

## Optional tools

These tools are optional, not required:

- NotebookLM: useful if you want source-grounded Q&A over uploaded course files.
- Perplexity: useful for quick sourced orientation, not final truth.
- Codex: useful for explaining or checking the optional Python script.
- Gemini Code Assist: useful for beginner coding help.
- GitHub Copilot: useful if you already use GitHub/VS Code.
- Cursor: useful if you want an AI coding editor.
- Claude Code: advanced coding-agent tool; not needed for beginners.

## Important rule

Use AI to help you think, structure, and check. Do not submit raw AI text as your own work. You must be able to explain the final result in your own words.
