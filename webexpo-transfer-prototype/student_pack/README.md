# Student pack export folder

This folder describes the downloadable student package for the activity **AI for student projects: from messy information to reliable decisions**.

The actual student package should be exported from the repo materials into a ZIP file or a Google Drive / Teams folder.

## Student pack goal

The package lets students complete the main activity without GitHub or coding tools.

Required student tools:

- Excel, Google Sheets, or LibreOffice Calc;
- one AI chat assistant such as ChatGPT, Gemini, or Claude;
- the provided CSV dataset.

Not required:

- GitHub;
- Python;
- Codex;
- Claude Code;
- GitHub Copilot;
- Cursor;
- paid tools.

## How to use it

1. Share the package as a ZIP file, Teams folder, or Google Drive folder.
2. Tell students to start with `00_START_HERE.pdf`.
3. Keep teacher notes outside the student package unless the lecturer wants them visible.
4. Keep technical materials clearly marked as optional.
