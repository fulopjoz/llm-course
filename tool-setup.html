Recommended by student type

No coding background

Use a spreadsheet plus ChatGPT, Gemini, or Claude. Do not start with GitHub, Python, Codex, or Claude Code.

Source-grounded work

Use NotebookLM if you want AI to answer from a controlled set of uploaded files.

Technical extension

Use Python and a coding assistant only if you already code or want to try.

Safe prompt pattern

Context: [what the project is about]
Data: [what columns or rows you are using]
Task: [what you want AI to do]
Constraints: [do not invent values; separate facts from assumptions]
Output format: [table, checklist, brief, questions]

Privacy rule

Never upload private, sensitive, patient, unpublished, or company-confidential data to a public AI tool. Use only the provided public sample dataset.
